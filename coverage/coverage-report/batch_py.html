18def batch(reader, batch_size, drop_last=False): 

19 """ 

20 This operator creates a batched reader which combines the data from the  

21 input reader to batched data. 

22  

23 Args: 

24 reader(generator): the data reader to read from. 

25 batch_size(int): size of each mini-batch. 

26 drop_last(bool, optional): If set to True, the last batch is dropped when  

27 the size of last batch is not equal to batch_size, if set to False, 

28 it will not. Default: False. 

29 Returns: 

30 The batched reader.  

31  

32 Return Type: 

33 generator  

34 

35 Examples: 

36 .. code-block:: python

38 import paddle 

39 def reader():

42 batch_reader = paddle.batch(reader, batch_size=2)

49 # [2, 3]

51 # [6, 7] 

52 # [8, 9] 

53 """
