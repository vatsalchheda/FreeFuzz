31class ParamAttr(object): 

32 """ 

33 

34 Note: 

35 ``gradient_clip`` of ``ParamAttr`` HAS BEEN DEPRECATED since 2.0.  

36 Please use ``need_clip`` in ``ParamAttr`` to speficiy the clip scope. 

37 There are three clipping strategies: :ref:`api_paddle_nn_ClipGradByGlobalNorm` ,  

38 :ref:`api_paddle_nn_ClipGradByNorm` , :ref:`api_paddle_nn_ClipGradByValue` . 

39 

40 Create a object to represent the attribute of parameter. The attributes are: 

41 name, initializer, learning rate, regularizer, trainable, gradient clip, 

42 and model average. 

43 

44 Parameters: 

45 name (str, optional): The parameter's name. Default None, meaning that the name 

46 would be created automatically. 

47 initializer (Initializer, optional): The method to initial this parameter. Default 

48 None, meaning that the weight parameter is initialized by Xavier initializer, 

49 and the bias parameter is initialized by 0. 

50 learning_rate (float, optional): The parameter's learning rate. The learning rate when 

51 optimize is the global learning rates times the parameter's learning rate times 

52 the factor of learning rate scheduler. Default 1.0. 

53 regularizer (WeightDecayRegularizer, optional): Regularization strategy. There are two method:  

54 :ref:`api_paddle_regularizer_L1Decay` , :ref:`api_paddle_regularizer_L2Decay` . If  

55 regularizer is also set in ``optimizer`` (such as :ref:`api_paddle_optimizer_SGD` ),  

56 that regularizer setting in optimizer will be ignored. Default None, meaning there is  

57 no regularization. 

58 trainable (bool, optional): Whether this parameter is trainable. Default True. 

59 do_model_average (bool, optional): Whether this parameter should do model average 

60 when model average is enabled. Only used in ExponentialMovingAverage. Default True. 

61 need_clip (bool, optional): Whether the parameter gradient need to be cliped in optimizer. Default is True. 

62 

63 Returns: 

64 ParamAttr Object. 

65 

66 Examples: 

67  

68 .. code-block:: python 

69 

70 import paddle 

71 

72 weight_attr = paddle.ParamAttr(name="weight", 

73 learning_rate=0.5, 

74 regularizer=paddle.regularizer.L2Decay(1.0), 

75 trainable=True) 

76 print(weight_attr.name) # "weight" 

77 paddle.nn.Linear(3, 4, weight_attr=weight_attr)

216class WeightNormParamAttr(ParamAttr): 

217 r""" 

218 

219 Note: 

220 Please use 'paddle.nn.utils.weight_norm' in dygraph mode. 

221  

222 Note: 

223 ``gradient_clip`` of ``ParamAttr`` HAS BEEN DEPRECATED since 2.0.  

224 Please use ``need_clip`` in ``ParamAttr`` to speficiy the clip scope. 

225 There are three clipping strategies: :ref:`api_paddle_nn_ClipGradByGlobalNorm` ,  

226 :ref:`api_paddle_nn_ClipGradByNorm` , :ref:`api_paddle_nn_ClipGradByValue` . 

227  

228 Parameter of weight Norm. Weight Norm is a reparameterization of the weight vectors 

229 in a neural network that decouples the magnitude of those weight vectors from 

230 their direction. Weight Norm has been implemented as discussed in this 

231 paper: `Weight Normalization: A Simple Reparameterization to Accelerate 

232 Training of Deep Neural Networks 

233 <https://arxiv.org/pdf/1602.07868.pdf>`_. 

234 

235 Args: 

236 dim(int, optional): Dimension over which to compute the norm. Dim is a non-negative 

237 number which is less than the rank of weight Tensor. For Example, dim can 

238 be chosen from 0, 1, 2, 3 for convolution whose weight shape is [cout, cin, kh, kw] 

239 and rank is 4. Default None, meaning that all elements will be normalized. 

240 name(str, optional): The parameter's name. Default None, meaning that the name would 

241 be created automatically. Please refer to :ref:`api_guide_Name` for more details. 

242 initializer(Initializer, optional): The method to initialize this parameter, such as 

243 ``initializer = paddle.nn.initializer.Constant(1.0)``. Default None, 

244 meaning that the weight parameter is initialized by Xavier initializer, and 

245 the bias parameter is initialized by 0. 

246 learning_rate(float32, optional): The parameter's learning rate when 

247 optimizer is :math:`global\_lr * parameter\_lr * scheduler\_factor`. 

248 Default 1.0. 

249 regularizer (WeightDecayRegularizer, optional): Regularization strategy. There are 

250 two method: :ref:`api_paddle_regularizer_L1Decay` , 

251 :ref:`api_paddle_regularizer_L2Decay`. 

252 If regularizer isralso set in ``optimizer`` 

253 (such as :ref:`api_paddle_optimizer_SGD` ), that regularizer setting in 

254 optimizer will be ignored. Default None, meaning there is no regularization. 

255 trainable(bool, optional): Whether this parameter is trainable. Default True. 

256 do_model_average(bool, optional): Whether this parameter should do model average. 

257 Default False. 

258 need_clip (bool, optional): Whether the parameter gradient need to be cliped in optimizer. Default is True. 

259 

260 Examples: 

261  

262 .. code-block:: python 

263  

264 import paddle 

265 

266 paddle.enable_static() 

267 

268 data = paddle.static.data(name="data", shape=[3, 32, 32], dtype="float32") 

269 

270 fc = paddle.static.nn.fc(x=data, 

271 size=1000, 

272 weight_attr=paddle.static.WeightNormParamAttr( 

273 dim=None, 

274 name='weight_norm_param', 

275 initializer=paddle.nn.initializer.Constant(1.0), 

276 learning_rate=1.0, 

277 regularizer=paddle.regularizer.L2Decay(0.1), 

278 trainable=True, 

279 do_model_average=False, 

280 need_clip=True)) 

281 

282 """
