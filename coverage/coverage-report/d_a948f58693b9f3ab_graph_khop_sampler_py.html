34 Graph Khop Sampler API. 

35 

36 This API is mainly used in Graph Learning domain, and the main purpose is to 

37 provide high performance graph khop sampling method with subgraph reindex step. 

38 For example, we get the CSC(Compressed Sparse Column) format of the input graph 

39 edges as `row` and `colptr`, so as to covert graph data into a suitable format 

40 for sampling. And the `input_nodes` means the nodes we need to sample neighbors, 

41 and `sample_sizes` means the number of neighbors and number of layers we want 

42 to sample. 

43 

44 Args: 

45 row (Tensor): One of the components of the CSC format of the input graph, and 

46 the shape should be [num_edges, 1] or [num_edges]. The available 

47 data type is int32, int64. 

48 colptr (Tensor): One of the components of the CSC format of the input graph, 

49 and the shape should be [num_nodes + 1, 1] or [num_nodes]. 

50 The data type should be the same with `row`. 

51 input_nodes (Tensor): The input nodes we need to sample neighbors for, and the 

52 data type should be the same with `row`. 

53 sample_sizes (list|tuple): The number of neighbors and number of layers we want 

54 to sample. The data type should be int, and the shape 

55 should only have one dimension. 

56 sorted_eids (Tensor, optional): The sorted edge ids, should not be None when `return_eids` 

57 is True. The shape should be [num_edges, 1], and the data 

58 type should be the same with `row`. Default is None. 

59 return_eids (bool, optional): Whether to return the id of the sample edges. Default is False. 

60 name (str, optional): Name for the operation (optional, default is None). 

61 For more information, please refer to :ref:`api_guide_Name`. 

62 

63 Returns: 

64 - edge_src (Tensor), The src index of the output edges, also means the first column of 

65 the edges. The shape is [num_sample_edges, 1] currently. 

66 - edge_dst (Tensor), The dst index of the output edges, also means the second column 

67 of the edges. The shape is [num_sample_edges, 1] currently. 

68 - sample_index (Tensor), The original id of the input nodes and sampled neighbor nodes. 

69 - reindex_nodes (Tensor), The reindex id of the input nodes. 

70 - edge_eids (Tensor), Return the id of the sample edges if `return_eids` is True. 

71 

72 Examples:
